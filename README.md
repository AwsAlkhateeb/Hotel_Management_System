# Hotel_Management_System

This project is a simple Hotel Management System developed using C# as part of the UML course.

Project Description:
The system is designed to manage basic hotel operations such as room reservations, guest management, services, and payments. The project is based on UML analysis and design principles.

Actors:
- Guest: Can reserve rooms, request services, make payments, and manage reservations.
- Manager: Can manage rooms, view reservations, guests, services, payments, and generate reports.

Technologies Used:
- C#
- Object-Oriented Programming (OOP)
- UML (Use Case Diagram and Class Diagram)

Notes:
This project focuses on system design and class structure rather than a full graphical user interface. The implementation follows the UML diagrams provided in the project.
